# ApexOS
What if your browser was an operating system? 
ApexOS is a browser-based operating system that replicates the desktop experience. It includes apps, windows, multitasking, and system tools. All these features run on the web.

## Features
- Fully interactive desktop environment
- Built-in applications and tools
- Integrated Music Player
- Task Manager for monitoring running apps
- Modern and responsive user interface
- Runs entirely in the browser with no installation required
- Completely Open Sourced

## 🛠️ Running Locally

### Requirements

- Node.js 18+
- npm

### 🛠️ Running Locally

```bash
git clone https://github.com/codingwithnsh/ApexOS.git

cd ApexOS

npm start
```

Open:

http://localhost:3000
